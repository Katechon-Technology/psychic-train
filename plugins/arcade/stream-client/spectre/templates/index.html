SPECTRE — PLATFORM REFERENCE

WHAT IS SPECTRE

Open-source global intelligence and threat monitoring dashboard. It fuses public OSINT into one map: geopolitical signals, conflict tracking, disasters, military air and satellite movement, strategic infrastructure, internet infrastructure disruptions, and GPS interference. Core data uses free/public sources only; no paid API keys are required for a full deployment.

DATA PIPELINE & RELIABILITY

  • Real-time updates: Server-Sent Events (SSE) endpoint /api/stream pushes data_updated events; the UI reloads data without manual refresh.
  • Fast boot: Two-phase load — /api/fast then /api/all — so the map becomes usable quickly while heavier domains finish.
  • Background refresh: Scheduled polling per domain (RSS, flights, satellites, quakes, blackouts, GPS jam, EONET, GDELT, Telegram).
  • Cache + resilience: Stale-while-revalidate behavior, disk cache (.spectre_cache.json) for warm restarts, graceful degradation if a source fails.
  • Fallback: If SSE is unavailable, periodic refresh still pulls updates.

SOURCE INGESTION

  • RSS — curated feeds in config.py.
  • Telegram — direct t.me/s/ channel scrape plus RSSHub fallback.
  • Geocoding — map search place names via Nominatim (OpenStreetMap), respecting reasonable request rates.

HEADER & STATUS

  • Top classification strip: SPECTRE OSINT DASHBOARD.
  • DEFCON STATUS badge — global threat level from aggregated OSINT severity.
  • Live counters: OSINT items, seismic events, natural events, military flights, net blackouts.
  • REFRESH line — transport state (e.g. LINK SECURE when synced).
  • ZULU TIME clock (UTC).
  • Lower header strip: system id (e.g. SPECTRE-2A), feed inventory, mode (LIVE CONTINUOUS), refresh cadence, and feed sources (RSS, TG-OSINT, USGS, EONET, GDELT, ADS-B, CelesTrak, IODA).

MAP OVERLAYS (layer bar)

Toggle each layer on the map. Color coding follows classification.

  • OSINT Nodes — RSS/Telegram-derived hotspots with geo.
  • GDELT Corpus — GDELT 2.0 conflict-related events (CSV export pipeline).
  • Theater Instability — country risk rings from OSINT-derived country risk.
  • Air Tracks — military-focused ADS-B tracks (ADS-B.lol).
  • Satellites — CelesTrak TLE + SGP4 propagation.
  • Net Blackouts — IODA internet outage alerts.
  • GPS Jamming · 24h — GPSJam H3 cells (high-interference snapshot; not real-time).
  • Bases — military base reference points.
  • Nuclear — declared / known nuclear facility reference data.
  • Weather — RainViewer global radar overlay (past ~2h; zoom limits per provider).
  • Seismic — USGS M4.5+ feed.
  • Nat. Events — NASA EONET natural events.

Keyboard shortcuts: keys 1–0 toggle layers in order (OSINT → … → natural events). Satellites and GPS jamming are not on number keys — use the layer bar buttons.

LEFT SIDEBAR

  • THREAT ASSESSMENT — counts for critical / high / medium / M6+ quakes; top signal categories from the classifier.
  • THEATER INSTABILITY — ranked country risk; click a row to filter the Live OSINT feed to that theater (country).
  • NET BLACKOUTS — IODA list; click an item to fly the map to that location.

MAP & NAVIGATION

  • Search — latitude/longitude, MGRS (via mgrs.js), or place name (Nominatim geocoding). Enter or GO.
  • COCOM quick-fly — GLOBAL plus regional buttons; Alt+1–6 maps to EUCOM through SOUTHCOM; G resets to global view.
  • Coordinate readout — MGRS under cursor while moving over the map.
  • Leaflet — zoom control (bottom-right), metric scale (bottom-left), Esri imagery base, boundaries/labels overlay; Weather layer adds RainViewer radar tiles when enabled.
  • Fullscreen map — map control button or F hides side panels for a clean map.
  • Right-click map — context menu: VIEW SATELLITE IMAGERY opens NASA GIBS imagery in a modal with date selection and a Leaflet preview.

RIGHT COLUMN — SEISMIC & OSINT FEED

  • SEISMIC LOG — recent quakes; click to fly to the epicenter.
  • LIVE OSINT FEED — severity-ranked items (critical→info), recency within tiers; 24-hour retention for high-signal feed.
  • Classifier — severities (critical, high, medium, low, info) and categories: conflict, terrorism, cyber, disaster, political, health, economic, nuclear, maritime, aerospace; deterministic color mapping on map markers and feed rows.
  • Country filter — banner when filtering from Theater Instability; CLEAR to reset.
  • Search — text filter in the feed list.
  • Filters — ALL, CRITICAL, HIGH, MEDIUM, KINETIC, CYBER, NUCLEAR, MARITIME, HEALTH, TG-OSINT (Telegram-tagged items).
  • Export — CSV or JSON snapshot of the current filtered feed (scope label shows FULL FEED or filtered). JSON includes export metadata (UTC timestamp, active filters, UNCLASSIFIED // OPEN SOURCE label).
  • NEW badges — newly ingested items since your session can show a NEW tag; toasts can alert on critical or bulk ingest.

FEED & MAP INTERACTIONS

  • Left-click OSINT item — menu: open source, copy OSINT snippet, copy URL, fly to location.
  • C — copy last left-clicked article to clipboard.

TOOLBAR

  • ? HELP — this reference and replay guided tour.
  • BROADCAST — curated MP3/HLS radio streams (HLS.js); draggable panel, volume, now-playing.
  • LIVE FEEDS — public camera / live iframe grid; region filters ALL / NEWS / URBAN.
  • SHORTCUTS — full keyboard map (also ?).

STATUS BAR

  • Per-source health dots: OSINT, USGS, EONET, ADS-B, Space, GDELT-CSV, IODA, TG-OSINT.
  • LAST SYNC timestamp (UTC-style).

LAYOUT & PERSISTENCE

  • Resize — drag left/right handles to widen side panels.
  • Panel collapse — click panel headers; panel collapse state persists in localStorage.
  • Toasts — transient notifications (e.g. fullscreen, sync, export).
